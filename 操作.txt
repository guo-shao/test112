git init   初始化一个仓库  生成.git文件(工作区)

创建用户名邮箱：
git config --global user.name ‘guo’
git config --global user.email ‘guo@.163.com’

将代码存储到.git仓库中
1.将代码放置到仓储的门口(缓冲区)
git add ./readme.txt
2.把仓储门口的的代码放到仓库中(版本库)
git commit -m ‘对添加的东西的说明’

git status 查看当前工作状态

git commit --all -m '说明'  一次性将所有文件添加到仓库(版本库)中，不用add

git log  查看历史提交日志 
git log --oneline  查看简洁版的日志

回退到指定的版本：
git reset --hard Head~1
指定版本号回退：
git reset --hard a97ed67

git reflog  每次对版本切换的记录


#####分支#####
默认是主分支master
创建dev分支：
	git branch dev
	在刚创建的dev分支里的东西和master分支里的东西是一样的
切换分支：
	git checkout dev
	切换到指定的dev分支
	git branch
	查看当前的分支
合并分支：
	先切换到主分支 git checkout master
	git merge dev
	当前分支与dev分支合并(有星号标记的是当前分支)
